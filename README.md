## ✨ 주요 기능 (Key Features)
<img width="1920" height="1080" alt="10_어바웃무무" src="https://github.com/user-attachments/assets/25066f10-8978-43a2-8f54-5fa6f4f00e11" />

* **통합 탐색:** 분산된 무료 교육 정보와 무료 공간(시설 대관) 정보를 한곳에서 검색.
* **지도 기반 검색:** 교육 및 공간 정보를 지도를 통해 시각적으로 탐색 (교육 지도, 공간 지도).
* **상세 필터링:**
    * **교육 정보:** 지역구, 분야, 모집 상태로 필터링.
    * **공간 정보:** 지역구, 대상자, 시설 유형으로 필터링.
* **반응형 UI/UX:** 데스크톱, 태블릿, 모바일 등 다양한 디바이스 환경 지원.
* **회원 기능:**
    * 카카오 소셜 로그인을 통한 간편한 회원가입.
    * 관심 있는 교육 및 공간 정보를 저장하는 **북마크 기능**.


## 🛠️ 기술 스택 (Tech Stack)
<img width="1920" height="1080" alt="6_서비스 구조도" src="https://github.com/user-attachments/assets/3a64e648-b470-44a3-b300-da49445bf3df" />
<img width="1920" height="1080" alt="7_서비스 아키텍처" src="https://github.com/user-attachments/assets/35f265b9-8e84-492a-95f0-64acec1973aa" />

| 구분 | 기술 |
| :--- | :--- |
| **Frontend** | React |
| **Backend** | Spring Boot, JPA |
| **Database** | AWS RDS (MySQL) |
| **Infra** | AWS EC2, Docker |
| **CI/CD** | GitHub Actions |
| **Data Update**| AWS Lambda, AWS EventBridge |
| **API Docs** | Swagger UI |

* **데이터 자동 업데이트:** AWS EventBridge와 Lambda를 사용해 공공 데이터를 주기적으로 AWS RDS에 업데이트하는 시스템을 구축했습니다.
* **개발 협업:** GitHub Flow 전략을 준수하고, GitHub Project 보드로 진행 상황을 시각화했습니다.
